# Loan Default Risk Analysis using Excel & Power Query

## Project Overview
This project analyzes loan default behavior using Excel, Power Query, Pivot Tables, and Pivot Charts.  
The objective was to identify high-risk customer segments and derive business insights that can support credit risk assessment and lending decisions.

---

## Dataset
- Total Records: 144,104
- Domain: Banking / Financial Risk Analysis
- Target Variable:
  - `Status = 1` → Default
  - `Status = 0` → Non-default

---

## Tools & Technologies Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboarding

---

## Data Cleaning & Preprocessing
Using Power Query:
- Handled missing values
- Corrected inconsistent formats
- Removed unnecessary data issues
- Prepared dataset for analysis

Feature engineering techniques:
- Income Grouping
- DTI (Debt-to-Income) Categorization
- Comparative Risk Segmentation

---

## Analysis Performed
The project analyzed loan default rates across:
- Age Groups
- Regions
- Occupancy Types
- Loan Categories
- Income Levels
- Debt-to-Income Ratios

---

## Key Insights

### 1. Commercial Loans Showed Higher Risk
Business/commercial loans had significantly higher default rates compared to regular loans.

### 2. Investment Properties Were Riskier
Investment residence borrowers showed higher default probability than primary residence borrowers.

### 3. High DTI Customers Defaulted More
Customers with higher debt-to-income ratios showed increased financial risk.

### 4. Regional Differences Were Observed
North-East region customers showed comparatively higher default rates.

### 5. Higher Income Customers Were Safer Borrowers
Customers with higher income levels showed lower default rates.
